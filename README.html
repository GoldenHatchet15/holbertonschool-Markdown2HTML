Markdown to HTML Converter

Description

Markdown is awesome! All your README.md are made in Markdown, but do you know how GitHub are rendering them?

Requirements

- All your files will be interpreted/compiled on Ubuntu 18.04 LTS using python3 (version 3.7 or higher) - The first line of all your files should be exactly #!/usr/bin/python3 - A README.md file, at the root of the folder of the project, is mandatory - Your code should use the PEP 8 style (version 1.7.*) - All your files must be executable - All your modules should be documented: python3 -c 'print(import("my_module").doc)' - Your code should not be executed when imported (by using if name == "main":) - You are not allowed to use pythonâ€™s library Markdown.
